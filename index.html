Разработка статического сайта

Создать логически связанную и завершенную информационную среду c использованием фреймов, таблиц или блоков (в соответствии с указанием преподавателя) и содержащую перечисленные ниже элементы.

  1. Единую для всех страниц навигационную часть, имеющую:
    1. текстовые и графические гиперссылки, открывающиеся в текущем окне :
    2. в том числе на закладку (ссылку на внутренний фрагмент страницы) одной из страниц
    3. на один или несколько внешних сайтов или страниц внешних сайтов, открывающихся в новом окне браузера
    4. для перехода на начальную страницу Вашей среды, открывающуюся в текущем окне браузера и
    5. показывающую посетителю его текущее положение выделением фрагмента навигации (выделение оформить с использованием таблицы стилей), пример оформления на http://198.211.123.137:7632/cgi-bin/py_sql_pages.py?function=page&page_id=20

  2. Информационные страницы (4-6), которые должны содержать:
    1. навигацию (раздел 1, раздел 2, к началу ) внутри страницы с использование закладок
    2. фрагменты, содержащие списки
      • нумерованные
      • не нумерованные
      • вложенные
    3. изображения объектов или субъектов с соответствующими пояснениями и комментариями
    4. изображение большого размера (600 пикселей по горизонтали и 400 по вертикали), открывающиеся в новом окне или закладке в центре как по вертикали, так и по горизонтали с использованием миниатюрной графической гиперссылки (размер по горизонтали до 100 пикселей)
    5. таблицу, содержащую: изображение на заднем фоне всей таблицы, изображение в одной из ячеек, изображение, гиперссылки

  3. Фрагменты с формами и данными должны содержать:
    1. cтраницу с формой для отправки информации на удаленный сервер контактных данных посетителя (фамилия, имя, мобильный телефон, электронка, дата отправки сообщения)
    2. cтраницу с формой для отправки на удаленный сервер результатов тестирований по HTML (только для БИ)
    3. страницу для отправки на удаленный сервер оригинальных данных (заказы из магазина, отзывы на что-либо, формулировка задачи и данные для решения задачи, или другое относящееся к тематике сайта?, названия переменных должны соответствовать задаче)(не обязательно)
    4. страницe для описания, решения и регистрации на сервере алгоритмической задачи (отправлять условие и результаты)(только для БИ)(не обязательно)
    5. работающие ссылки на файлы с отправляемыми данными, ссылки на файлы могут быть размещены на страницах с формами и должны открываться в новом окне браузера.
